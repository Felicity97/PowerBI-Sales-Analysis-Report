# Power BI Sales Analysis Report

This Power BI report analyzes sales performance across categories, segments, and regions.  
It includes:
- KPI cards (Total Sales, Profit, Quantity)
- Sales by Sub-category
- Regional performance map
- Trend by Order Date

**How to use:**
1. Download the `.pbit` file.
2. Open it in Power BI Desktop.
3. Load your own sales dataset to generate insights.
